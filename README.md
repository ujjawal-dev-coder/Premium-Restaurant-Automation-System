A high-end, futuristic restaurant website built with a focus on **Obsidian Dark Aesthetics** and seamless user experience. This project features a fully functional automated ordering system with dual-mode fulfillment.

---

## ✨ Key Features

- **💎 Glassmorphic UI:** Premium dark theme using Pure Obsidian Black with blurred glass card effects.
- **🛒 Dynamic Cart System:** Real-time price calculation and item management.
- **📦 Intelligent Fulfillment Popup:**
    - **Dine-In:** Allows customers to select specific table numbers.
    - **Home Delivery:** Integrated address collection for off-site orders.
- **🚀 Automation:** Powered by **FormSubmit** to send orders directly to the restaurant's email without a backend.
- **🎭 Smooth Animations:** Powered by **GSAP (GreenSock)** for hero transitions and scroll-triggered reveals.
- **📱 Responsive Design:** Fully optimized for mobile, tablet, and desktop screens using **Tailwind CSS**.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS
- **Interactions:** GSAP (GreenSock Animation Platform)
- **Typography:** Plus Jakarta Sans & Playfair Display
- **Order Handling:** FormSubmit API
- **Icons:** Lucide Icons / SVG

---

## 📸 Design Philosophy

The project follows a **Senior-level design approach**:
- **Color Palette:** `#0B0B0B` (Obsidian), `#FF8C00` (Accent Orange), and `#D4AF37` (Gold).
- **Layout:** Bento-grid style menu and minimalist navigation.
- **Experience:** Floating animations and high-quality image masking for a luxury feel.
